# Radhas Textiles Website

## 📌 Project Description

Radhas Textiles is a static business website developed to establish an online presence for a textile store. The website provides essential business information, enables customers to contact the store through WhatsApp, and helps visitors locate the store using Google Maps.

The project focuses on creating a clean, responsive, and user-friendly interface using front-end web technologies.

---

## 🚀 Features

- Responsive design for mobile, tablet, and desktop devices
- Business information and store details
- Direct WhatsApp contact integration
- Google Maps location integration
- Easy-to-use navigation menu
- Fast-loading static website
- Clean and modern user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
